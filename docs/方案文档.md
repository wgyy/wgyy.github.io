# 键盘云驱 - 技术方案文档

## 一、产品概述

### 1.1 产品定位

键盘云驱是一款基于Web的键盘配置管理工具，用户可通过浏览器直接对键盘进行键位映射、宏命令编辑、灯效调节等个性化配置。

### 1.2 核心特性

- **纯Web应用**: 无需安装桌面客户端，浏览器即可使用
- **多模连接**: 支持USB有线连接和蓝牙无线连接
- **本地存储**: 配置数据保存在浏览器本地，无需账户登录
- **可视化编辑**: 直观的键盘布局编辑器，所见即所得

## 三、功能模块

### 3.1 设备连接模块

#### 功能列表
- 自动检测已连接的USB设备
- 扫描并连接蓝牙设备
- 设备连接状态显示
- 断开连接操作
- 多设备切换

#### 技术实现
- WebUSB API: `navigator.usb`
- WebBluetooth API: `navigator.bluetooth`

### 3.2 键位映射模块

#### 功能列表
- 可视化键盘布局显示
- 按键点击选择
- 键位功能编辑（普通键/功能键/媒体键/宏/层切换）
- 多层管理（FN层、MO层等）
- 键位配置导入/导出

#### 技术实现
- SVG/Canvas渲染键盘布局
- 键盘布局JSON配置不同配列

### 3.3 宏管理模块

#### 功能列表
- 宏命令列表展示
- 宏录制功能
- 宏编辑（添加/删除/调整动作）
- 宏动作类型：按键、延迟、循环
- 宏与按键绑定

#### 技术实现
- 事件录制: 捕获keydown/keyup事件
- 宏存储结构: `Macro { id, name, actions: MacroAction[] }`

### 3.4 灯效控制模块

#### 功能列表
- 预设灯效选择（常亮/呼吸/波浪/雨滴等）
- 自定义颜色设置
- 亮度调节
- 速度调节
- 方向设置

#### 技术实现
- 颜色选择器组件
- 灯效参数配置面板

### 3.5 配置管理模块

#### 功能列表
- 配置保存到本地
- 配置导入/导出（JSON文件）
- 配置重置
- 配置历史记录

#### 技术实现
- LocalStorage / IndexedDB存储
- JSON文件读写

## 四、数据结构

### 4.1 设备状态

```typescript
interface DeviceState {
  connected: boolean;
  connectionType: 'usb' | 'bluetooth' | null;
  deviceInfo: {
    name: string;
    vendorId: number;
    productId: number;
    serialNumber: string;
  } | null;
  firmwareVersion: string | null;
}
```

### 4.2 键位配置

```typescript
interface KeymapConfig {
  layout: string;           // 键盘配列
  layers: {
    id: number;
    name: string;
    mappings: Record<string, KeyAction>;
  }[];
}

interface KeyAction {
  type: 'key' | 'fn' | 'macro' | 'layer' | 'none';
  value: string | number;
}
```

### 4.3 宏配置

```typescript
interface Macro {
  id: string;
  name: string;
  actions: MacroAction[];
}

interface MacroAction {
  type: 'press' | 'release' | 'delay';
  keycode?: number;
  duration?: number;  // 延迟毫秒
}
```

### 4.4 灯效配置

```typescript
interface LightingConfig {
  enabled: boolean;
  brightness: number;      // 0-100
  effect: LightingEffect;
  customColor?: {
    r: number;
    g: number;
    b: number;
  };
}

type LightingEffect = 'static' | 'breathing' | 'wave' | 'rain' | 'reactive' | 'off';
```

## 六、兼容性说明

### 6.1 浏览器支持

| 特性 | Chrome | Edge | Firefox | Safari |
|------|--------|------|---------|--------|
| WebUSB | 54+ | 79+ | 不支持 | 不支持 |
| WebBluetooth | 56+ | 79+ | 不支持 | 不支持 |

### 6.2 限制说明

- WebUSB/WebBluetooth需要在HTTPS环境下使用
- 蓝牙功能在macOS/iOS上有较多限制
- 部分功能需要用户手动授权

## 八、注意事项

1. **安全限制**: WebUSB/WebBluetooth需要在安全上下文（HTTPS或localhost）下运行
2. **设备协议**: 需要与键盘固件侧约定通信协议格式
3. **浏览器兼容**: 建议主要针对Chrome/Edge浏览器开发
4. **离线支持**: 可考虑添加PWA支持实现离线使用
